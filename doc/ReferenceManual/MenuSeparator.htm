Menu Separator

This trait allows you to introduce menu separator (horizontal bars) into a game piece's right-click menu.  A keystroke need not be provided if the separator does not need the ability to be hidden.

Separator description: Optional description of separator - no other effect.

Keystroke: If the specified keystroke is hidden (by a Restrict Commands trait), then this separator will be hidden also.
